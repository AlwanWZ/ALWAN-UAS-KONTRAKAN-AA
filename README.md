# ALWAN-UAS-KONTRAKAN-AA

Aplikasi manajemen kontrakan berbasis Next.js, Firebase, dan MongoDB.  
Dibuat untuk memenuhi tugas UAS Mata Kuliah Pemrograman Visual.

## Fitur Utama
- Login & Register (Firebase Auth)
- Dashboard untuk Admin, Penyewa, dan Tamu
- Manajemen Data Kontrakan, Pembayaran, dan Booking
- Upload gambar ke Cloudinary
- Firebase + MongoDB (via API)
